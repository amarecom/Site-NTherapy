# Site-NTherapy - Instructions du Projet

Ce projet consiste à créer un site internet moderne, performant et élégant pour un ostéopathe (NTherapy). Le site doit être parfaitement optimisé pour mobile et PC, avec une esthétique soignée inspirant confiance et professionnalisme.

## Objectifs Design & UX
- **Esthétique Moderne** : Utilisation de palettes de couleurs apaisantes (bleus doux, blancs, gris clairs), typographie lisible et espacement généreux.
- **Mobile-First** : Navigation fluide et tactile, temps de chargement ultra-rapides.
- **Confiance** : Sections dédiées aux témoignages, aux diplômes et à l'approche thérapeutique.
- **Conversion** : Boutons d'appel à l'action (CTA) clairs pour la prise de rendez-vous.

## Stack Technique
- **Framework** : React (TypeScript) pour une structure robuste et maintenable.
- **Stylisation** : Vanilla CSS (ou CSS Modules) pour une flexibilité maximale et une performance optimale sans dépendances lourdes.
- **Animations** : Transitions douces pour améliorer l'expérience utilisateur (Framer Motion ou CSS transitions).

## Structure du Site (Planifiée)
1. **Hero Section** : Présentation immédiate, valeur ajoutée et bouton de réservation.
2. **Services** : Détail des types de consultations (ostéopathie générale, sport, pédiatrie, etc.).
3. **À propos** : Parcours de l'ostéopathe, philosophie de soin.
4. **Témoignages** : Preuve sociale pour rassurer les patients.
5. **Infos Pratiques** : Carte, adresse, horaires et accès.
6. **Contact / Réservation** : Formulaire simple ou intégration d'un outil de réservation externe.

## Mandats de l'Agent
- Respecter scrupuleusement les standards d'accessibilité (WCAG).
- Optimiser le SEO local pour attirer des patients dans la zone géographique concernée.
- Toujours valider les modifications via des tests de rendu.
